Política de Privacidade

Robuuz - Hábitos & Dopamina

Última atualização: junho de 2026

1. Coleta de Dados

O Robuuz não coleta, armazena ou transmite nenhum dado pessoal para servidores externos. Todos os dados inseridos no app (hábitos, receitas, finanças) ficam armazenados exclusivamente no dispositivo do usuário.

2. Dados de Pagamento

As assinaturas Premium são processadas exclusivamente pelo Google Play. O Robuuz não tem acesso aos dados de pagamento do usuário.

3. Uso Offline

O app funciona 100% offline. Nenhuma informação pessoal é enviada para servidores.

4. Permissões

O app solicita apenas as permissões necessárias para o funcionamento das assinaturas via Google Play Billing.
